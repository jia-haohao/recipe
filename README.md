  .

## 就業 Term の技術

- devise
- rails_admin
- cancancan
- Ajax を使ったコメント機能
- お気に入り機能

## カリキュラム外の技術

- ransack

## 実行手順

```
$git clone https://github.com/jia-haohao/recipe.git
$cd recipe
$bundle install
$yarn install
$rails db:create
$rails db:migrate
$rails s
```
